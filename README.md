# worgodjs
I want to see the context that words are used in.

# Features
- Parses text.
- Shows word-count / frequency list after parsing.
- Ability to select a word from list and see the sentences that it belongs in.

## How To Use
```node userside.js```

```node wordgod.js filename.txt```
